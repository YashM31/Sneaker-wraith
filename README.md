# Sneaker Wraith

A simple sneaker showcase and AI price predictor web app with a clean white-and-black theme.

## How to run

1. Install dependencies:
    pip install -r requirements.txt

2. Start the server:
    uvicorn main:app --reload

Open your browser at [http://localhost:8000](http://localhost:8000)

- Click any sneaker image to see all details in a modal popup.
- Inspired by VegNonVeg grid style. All data & predictions are AI-powered.
